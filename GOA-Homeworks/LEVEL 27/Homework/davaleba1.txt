თქვენი სიტყვებით ახსენით რა არის დეკლარაცია, კუთვნილება, მნიშვნელობა, css rule, selector, css inheritance, descendant selector


1)CSS-ში დეკლარაცია განსაზღვრავს, რომ ელემენტი იყოს გასტილული.(გალამაზებული).
2)css-ში კუთვნილება არის მაგ: font-size.
3)css-ში მნიშვნელობა არის მაგ: font-size: 16px; 16px არის მნიშვნელობა.
3)css rule არის მაგ:

 p {
  color: blue;
  font-size: 14px;
}

აქ p არის სელექტორი, ხოლო ფრჩხილებში მოცემულია დეკლარაციები.

4)css-ში selector არის მაგ: 

h1 {
  color: green;
}

აქ h1 არის სელექტორი

5) css inheritance არის მემკვიდრეობა: მაგ; არსებობს რამდენიმე ისეთი თეგები სადაც რამდენიმე თეგის ჩაწერა შეგვიძლია მაგ:

<P> ჩვენ P-შეგვიძლია ჩავწეროთ ბევრი რამ მაგ: <B> , <U> , და ა.შ.

6)descendant selector- ეს სელექტორი არის შვილეული მაგ; პარაგრაფში რომ ჩავწეროთ <B> თეგი <B> თეგი გამოდის შვილეული.
